# Ek Paul Foundation - Frontend (React + Bootstrap)

## Quick start
1. unzip or clone the project
2. run `npm install`
3. run `npm start`
4. open http://localhost:3000

This is a static React frontend scaffold with Home, About, Events, and Donation pages.
Videos and images are referenced from free public URLs (placeholders). Replace them with your own assets in `public` or `src/assets` as needed.
